Problems Logging In

The corporate Active Directory service is used.

Users need to be members of the group: ‘CN=Asset Reliability Officers,OU=General Distribution Lists,OU=Mail Enabled,DC=energex,DC=com,DC=au.’ If you are being denied access to C.A.C, you will need to make a SPARQ service desk request to become a member of this group.

Auditing

C.A.C maintains audit logs of all changes made. At the end of each month these are emailed to the C.A.C-specific C.A.C AUDIT LOGS distribution list. If you need access to these logs, you should contact the SPARQ service desk.

Other

The 'About' link will bring up a new window that shows a fair amount of technical information regarding C.A.C. This may help the C.A.C support staff identify any problems, if such do occur.
